# ArchMove Landing Page

## How To Use :
 1. Install NodeJS.
 2. Install the dependencies:
    ```bash
    # Using npm
    npm install
    ```
 3. Start the development server:
 4. 
    ```bash
    # Using npm
    npm run dev
    ```

    Now you should be able to see the project running at localhost:3000.
    Or you can use live Sever Extension of vsCode.
